# Demo: Streamlit Dashboard

Visual control panel for monitoring NiFi data pipelines.

## Features

- **Control Panel**: Start/stop data generation
- **Data Monitor**: Real-time statistics and charts
- **Data Explorer**: Browse Snowflake tables
- **Service Manager**: View SPCS service status

## Deploy

```bash
./deploy.sh
```

## Access

After deployment, access the Streamlit app from Snowsight:
1. Go to Snowsight
2. Navigate to Streamlit Apps
3. Open "NIFI_DEMO_DASHBOARD"
